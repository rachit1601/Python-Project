# Trivia Quiz
Its just a simple trivia quiz program.

## Requirements
>pip , 
>pygame ,
>ttkthemes

## Features
It has many features:
1) Ever expanding question databse
2) MCQ/ True-false Questions
3) 2 background Music
4) Your Stats
5) More than 5 categories
6) 1 offline quiz in each category
7) Many Themes 

And much more...
